## Pocket Clarity

Pocket Clarity treats the extension as a small, precise utility at the edge of the browser. The visual system is quiet and practical: broad browser context, one focused side panel, and a compact icon that acts as the anchor.

The composition should make the workflow obvious without heavy explanation. Saved text, links, and images become stacked cards; search and selection appear as calm, organized states. Color stays restrained: graphite for structure, copper from the icon for identity, sky blue for active interaction.

Text is sparse and functional. The screenshots should feel like polished product evidence, not a banner ad. Every element needs deliberate spacing, clean hierarchy, and enough breathing room to remain readable at Chrome Web Store preview sizes.
